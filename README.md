# Описание проекта

### Серверная часть мессенджера
```server_messenger/``` 
### Клиентская часть мессенджера
```client_messenger/``` 

# Установка
## Установка серверной части
```bash
cd server_messenger
build
```
## Конечные папки
```server_messenger/bin```
```server_messenger/lib``` 

## Установка клиентской части
```bash
cd client_messenger
build
```
## Конечные папки
```client_messenger/bin```
```client_messenger/lib```
